API reference
All endpoints — test with Postman or curl
MethodEndpointAuth?Description
POST/api/auth/registerNoRegister new user
POST/api/auth/loginNoLogin, get JWT token
GET/api/wallet/balanceYesGet current balance
POST/api/wallet/depositYesTop up wallet
POST/api/transactions/sendYesSend money to user
GET/api/transactions/historyYesList all transactions
GET/api/transactions/:idYesGet single transaction
GET/api/healthNoHealth check
Example: send money
curl -X POST http://localhost:5000/api/transactions/send \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "receiver_phone": "0722000001",
    "amount": 500,
    "description": "Lunch money"
  }'
